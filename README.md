
# 📊 Analyse des ventes – Projet Business Analyst (Excel)

## 🏢 Contexte

Ce projet analyse les données de vente d'une entreprise de distribution 
pour identifier les tendances, évaluer les performances commerciales 
et formuler des recommandations concrètes.

**Outils utilisés :** Excel — nettoyage des données, KPIs, 
analyse exploratoire et tableau de bord interactif.
